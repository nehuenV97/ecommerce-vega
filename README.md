## Tecnologías utilizadas:   
**HTML5:** Utilizado para la estructura de las páginas del sitio.   
**CSS3:** Se utilizó para aplicar estilos personalizados.  
**JavaScript:** Lenguaje principal para agregar interactividad. Manejo del DOM. Interacción con el navegador y agrega funcionalidades.
**JSON:** Utilizada para almacenar y organizar todos los productos del catálogo.   
**Session y LocalStorage:** Persistencia de datos en el carrito y en el registro de usuario.

## Roadmap:   
### Etapa 1: Estructura general de la aplicación 
- Elección de nombre y tipo de e-commerce
- Creación de la estructura principal con html
- Creación de las páginas en html por donde se navegará (notebooks, celulares, accesorios)
- Estructura de los formularios de login y registro
- Creación de un logo
  
### Etapa 2: Estilos con css
- Utilización de css para dar estilo e identidad a la aplicación
- Estructuración y estilización del navbar
- Estructuración y estilización de los formularios (login y registro)
- Estructura de una card de producto

### Etapa 3: Javascript
- Javascript para redireccionar al usuario
- Redirección desde el login a la página principal (una vez logueado el usuario)
- Botón logout, redirección hacia el login (cierre de sesión)
- Componente navbar en javascript
- Crear estructura del componente card en javascript

### Etapa 4: JSON de productos
- Crear la estructura JSON de los productos
- Utilización de FETCH para recuperar información desde el JSON
- Rellenar las cards con la información de cada producto del JSON
- Filtrar las páginas por categoría, cada página muestra sus productos correspondientes

### Etapa 5: LocalStorage y SessionStorage
- Uso de Session y LocalStorage para la persistencia de datos
- sessionStorage para guardar información del usuario logueado
- localStorage para guardar cada producto que el usario quiera comprar
- Creación de la página carrito de compras para mostrar productos
- Botón para eliminar productos del carrito

## Mejora Significativa
- Refactorización de código.  
- Aplicación responsive y mejora visual.  
- Implementación de alertas y mensajes de confirmación.
- Implementación de los botones vaciar carrito y finalizar compra en el carrito
